# 回溯算法

参考组合问题：

![image-20210506141448598](https://gitee.com/HappyBinbin/pcigo/raw/master/pic/20210506141448.png)

```java
void backtracking(参数) {
    if (终⽌条件) {
        存放结果;
        return;
    }
    for (选择：本层集合中元素（树中节点孩⼦的数量就是集合的⼤⼩）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```

## 剪枝优化
