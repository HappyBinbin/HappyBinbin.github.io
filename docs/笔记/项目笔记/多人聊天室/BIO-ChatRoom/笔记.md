# BIO

 BIO：同步阻塞

那么它阻塞在什么地方呢？

- ServerSocket().accept()
    - 服务器线程一直在等待用户的连接
- InputStream().read / OutputStream.wirte()
    - 如果客户端一直没有输入，那么服务器为其建立的Handler的线程就会一直等待，被浪费
- 无法在同一个线程里处理多个Stream I/O
